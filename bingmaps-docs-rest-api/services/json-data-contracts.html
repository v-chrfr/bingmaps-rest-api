JSON Data Contracts

Important

Update: Since writing this documentation the Bing Maps team has released an open source portable class library which makes it easy to access the Bing Maps REST services from any .NET application. You can find this library on GitHub here. Going forward the data contracts on this page will no longer be updated. Please refer to the .NET REST Toolkit for the latest JSON contrcts.

The following data contracts include classes for all Bing Maps REST Services. For examples of how to use these data contracts, see Using the REST Services with .NET.
